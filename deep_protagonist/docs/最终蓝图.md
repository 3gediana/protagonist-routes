# 最终蓝图（北极星）

> 最后更新 2026-05-24（**v2 二次升级 D-034**：加入完整生存模拟主线）
> 这是 deep_protagonist 的"目的地"。每个阶段决策都要对齐它。

## 一句话

把一个原始人胶囊扔进 1024m 真自然世界，让他**自学求生 + 进化文明三阶段**：
- **早期**（裸奔 → 草衣）：采草造衣、采果充饥、找水、探索
- **中期**（穴居 → 安居）：真实建造（放图纸→堆材料→自动盖房，4 种房型）
- **后期**（采集 → 定居）：农耕（种田收果实）、饲养（围栏关动物喂草繁殖）、驯服（与动物建立 trust，让其跟随）

## 答辩 demo 用户看到什么

启动 → 一片 **1024×1024 米的真实自然** → 一只原始人胶囊在野外活动。

观察画面（按 progression 阶段顺序）：

### 早期（Early Stage）
- **饿了找食物，渴了找水，累了找庇护**
- **狩猎**：追兔子、伏击鹿、躲狼群、河里抓鱼
- **采集**：果树下捡果、浆果灌木、仙人掌（沙漠续命）、蘑菇（小心中毒）
- **造衣服**：拾长草 → 编草裙（抗冷 +10%），杀兔鹿后兽皮 → 兽皮护寢（抗冷 +30%）
- **空间记忆**：知道家、水源、狼出没区在哪
- **地形适应**：沙漠快速失水、沼泽减速、山顶视野好
- **昼夜**：天黑找洞穴/林地躲，狼夜行更凶
- **工具**：木棒+石尖=长矛，狩猎成功率倍增

### 中期（Mid Stage，解锁条件：采集 30 资源 + 探索 ≥500m）
- **真实建造**：选房型（小棚/木屋/石屋/大屋）→ 在地上放"图纸"白框 → 反复搬木头石头草放上去 → 系统数够材料 → 自动盖好真 3D 房子
- **房子带功能**：屋内庇护（防狼/防雨/恢复体温）+ 储物（无限堆放材料）
- **简易庇护所降级**：早期手搭的 lean-to 仍能用，但中期开始 agent 应该升级

### 后期（Late Stage，解锁条件：建造完一座房子）
- **农耕**：种子（吃 fruit 时掉落）→ 种地 → 浇水 → 长大 → 收果实
- **饲养**：抓兔关木栅栏 → 喂草 → 繁殖出小兔
- **驯服**：长期喂食单只狼/鹿 → trust 累积 → 让其跟随 agent
- **应对突发**：用户按键扔狼/起火/降雨，agent 实时反应（S6）

### 主线核心机制
- **progression gating**：动作按阶段解锁，agent 必须按"采草→建造→农耕"顺序学
- **课程学习 reward**：每完成一阶段 +50 大奖励，引导 PPO 一阶段一阶段收敛
- **EWC 防遗忘**（蓝图原则 line 87）：学新阶段不忘旧阶段

## 世界

| 类别 | 内容 |
|---|---|
| 尺寸 | 1024 × 1024 米 |
| 地形分块 (biomes) | 平原 / 树林 / 山脉 / 河流 / 沼泽 / 沙漠 |
| 动物 | 🐰兔 🦌鹿 🐺狼 🐟鱼 🦅鹰（鹰只装饰） |
| 植被 | 果树 / 浆果灌木 / 仙人掌 / 蘑菇 |
| 资源 | 干柴 / 石头 / 长草（搭窝） |
| 时间 | 昼夜循环（约 10-15 分钟一天） |

地形不手画，用噪声+海拔+湿度自动分块（procedural biomes）。

## agent 生命系统

- 饥饿（0-100）：0 死亡
- 口渴（0-100）：0 死亡
- 体力（0-100）：累了走慢
- 体温（0-100）：寒夜不烤火会扣血
- 持有物品：0-3 件简单物品（木棒、石尖、长矛、干柴）
- 庇护所位置：可记 1 个

## agent 神经网络

- **一只**网络从头训到尾，参数永不重置
- 起步 5-10M 参数（S4），最终 50-60M+ （S5）
- libtorch + PPO + RNN（带短期记忆）
- 长期记忆通过额外的 spatial memory module（S5+）

## 阶段路线（按 AI 编码速率换算）

> 换算规则：AI 写 30 分钟 ≈ 程序员 1 个月。开发时间被极大压缩。
> 真正的硬约束是 **GPU 训练时长**（机器在跑，无法压缩）。

| | 内容 | 开发 (AI) | 训练 (GPU) |
|---|---|---|---|
| S0 | ✓ 项目骨架 + libtorch + CUDA | 完成 | — |
| S1 | ✓ 3D 高度图世界 + 洞穴 + 物理 | 完成 | — |
| **S2** | 1024m 大世界 + 6 生物群系 + 河流 + 昼夜 | 1-3 天 | — |
| **S3** | 动物 + 植被 + 资源 + agent 生命状态 | 1-3 天 | — |
| **S4** | PPO 训练（觅食+喝水+躲狼）+ 5-10M 模型 | 1-2 天 | 1-2 周 |
| **S5** | 工具 + 生火 + 庇护所 + 扩到 50M + 长训 | 3-5 天 | 3-4 周 |
| **S6** | 用户干预 + 答辩预演 + 调参 | 1-2 天 | — |

**总周期：约 4-8 周**（其中开发约 1-2 周，训练约 4-6 周可挂机）

注：你随时可中断训练（checkpoint+resume），所以"4-6 周训练"不是连续锁定的

## 不做的事（锁死，免范围扩散）

- ❌ 多只 agent / 群体 / 部落（那是 protagonist_ecology 的事）
- ❌ 烹饪复杂度 / 多语言（保留单一菜系，无对话语义）
- ❌ 复杂物理（不写流体、布料、软体）
- ❌ 全面真实美术（地形/动物/植物仍是几何，但**房子破例用 Quaternius GLB**——S3.10 必需）
- ❌ 多人对话 / NPC 智能（NPC 全是脚本）

> **D-034 升级（2026-05-24）**：原蓝图禁止"农业/衣物"，二次升级后**允许**农耕/衣服/饲养/驯服三件，作为后期 progression 的解锁内容。详见 `docs/决策记录.md` D-034。

## 单 agent 训练原则（锁死）

- **同一个神经网络从 S4 训到 S6**，参数永不重置
- **每加新任务，旧奖励权重保留**（防遗忘）
- 网络输出包括所有动作（基础 + 工具 + 建造），即使早期某些动作没意义
- 训练曲线分别记录每种行为的得分，**任一退化立刻警报**
- 必要时上 EWC（弹性参数固化）防遗忘

## 答辩故事

> 我没有手动给 agent 写"看到狼就跑"的代码。它**自己学会**了在大自然里求生：
> - 学会先采集、再狩猎
> - 学会有工具的话先做工具
> - 学会昼夜分配活动
> - 学会避开危险地形
>
> 这一切只来自一个奖励：**活下去**。
