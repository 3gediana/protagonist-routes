## 4. 全程阶段路线（目标 / 做什么 / 成功判据 / 表现好→下一步）

### S 提速（跨阶段加速器，最先做）
- 目标：脑子前向从单线程铺到多核，接近核数倍加速。
- 做什么：在共享引擎 **AgentLayer 决策调度层**，把同 tick 内 N 个**互相独立**的 per-agent 前向并行（**读全部观测 → 并行算 → 按固定 agent index 顺序统一写回**）。数学等价、不动数据流、非乱并发。
- 成功判据：8 代 smoke **harmony 7/7** + gen0/gen7 fitness 轨迹在多线程噪声带内（参考 base vs base2 ~0.8% 偏移）+ wall-time 明显下降 + CPU 核占用从 ~1 升到多核。
- 表现好→下一步：commit，回拟真主线 P1。出问题→§5.0。

### A 尺度（温和版，按需，不追上千实体）
- 目标：世界/agent/单局适度放大（上限 ~900×900 / ~40-50 agent / ~1200 tick）。
- 成功判据：8G 显存不爆 + 80 代 ≤8h + harmony 7/7。
- 表现好→B。出问题（爆显存/超时）→§5.1。

### B 环境真实压力（当前阶段）
- 目标：季节/天气/温度真的会冻会饿，逼 agent 用环境信息。
- 做什么：① ON/OFF 对照收官(P1)；②若证明感知**被用上**→保持；若**没被用上**→逐档加压（降体温下限 / 升冷阈值 / 冬季食物枯竭），每档 smoke 防全灭，直到"看天行事"成为活下去的必需。
- 成功判据：harmony 7/7 + 出现"避寒/囤粮"适应行为 + 背景种群季节性减员但**不灭绝**。
- 表现好→C。出问题（全灭/harmony 破）→§5.2。

### C 局部感知

- 表现好→D。出问题→§5.3 / §5.4。

### D 连续世代 + 血缘
- 目标：年龄 / 亲代抚育 / 亲缘识别 / 世代重叠，取代离散代际进化。

## 5. 决策树（出问题怎么办——全部预先决策，**别停下问用户**）

### 5.0 提速并行化出问题
- **结果跑偏**（超噪声带 / harmony 破）：多半是并行写回顺序错 → 改成"并行**只读+算**，写回严格按 agent index **串行**"。仍偏 → **回退该改动**，记录负结果（铁律 3），回拟真主线，不纠缠。
- **没加速**（核没用满）：查热路径有无全局锁（GPU stream mutex / 共享 RNG / 世界锁）→ 锁在热路径就用 per-thread 副本消除；改不动 → 回退，标记"引擎级限制"，继续拟真主线。
- **编译失败/崩溃**：先回退到能跑的版本保命，再小步重试。

### 5.1 尺度爆显存 / 超时
- 爆 8G 显存 → 缩 agent 数或世界尺寸一档重试；仍爆 → 放弃放大（用户不追上千实体），保持当前尺度。
- 超 8h → 降代数压进 8h。**拟真优先于尺度**。

### 5.2 加压力后全灭 / harmony 破
- 全灭(alive=0) → 压力档**回退一档**（经验：dmg=0.5 太狠会全灭 / dmg=0.25 是 Goldilocks），重 smoke。
- harmony 某条破，分情况：C7 灭绝 = 压力太狠，回退；C6 多样性低 = 多半短跑没收敛，**跑满代再判**；C1-C5 涌现行为破 = 压力可能扭曲生态，回退并根因诊断。

### 5.3 改输入维后旧 checkpoint 不兼容
正常现象（shape guard 正确拒绝）→ **起新血脉从 gen0 跑**，不接力。等拟真架构定型再开接力链（`runtime.resume_checkpoint_path`）。

### 5.4 指标长期低
先怀疑算法/reward/代码 bug → 根因诊断 → 系统性修。**绝不**调小参数刷指标、**绝不**为某指标加直接奖励（Goodhart，铁律 2）。

### 5.5 GPU 被 deep_protagonist 占用
等它空出来再跑（单卡 8G，俩训练抢显存都跑不好）；期间做**不占 GPU**的活（读/改代码、写文档、profile 分析）。**绝不和它抢**——deep_protagonist 是大头（练 50MB 模型、一轮两三天），优先级更高。

### 5.6 trace 堆积 / 磁盘满
跑完**立刻删 trace**，只留分析产物（.md/.json 小结）。
